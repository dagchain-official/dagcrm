04

Save karte hi khud‑ba‑khud hisaab

Ye formulas har save pe apne aap chalte hain — tum type nahi karte.

Revenue

On save

Net revenue = Gross − Commission — automatic.

Payroll

On save

Final salary = Basic + Incentive + Bonus − Deduction.

Proposal item

On save

Amount = Qty × Price × (1 − Discount%); total/tax bhi auto.

Conversion date

On save

Lead Converted hote hi converted date stamp — isi se mahine‑wise KPI count hoti hai.

06

Paisa store karne ke liye button dabana zaroori

Incentive aur Payroll apne aap post nahi hote. Admin / Finance / HR ko "Run" dabana padta hai — tabhi number save hota hai.

Incentive / Formula Run

Run dabao

Slab ya formula se incentive nikaal kar Incentive + Payroll.incentive me likhta hai → final salary dobara ban‑ti hai.

Preview boards sirf dikhate hain; jab tak Run nahi dabate, payroll me kuch nahi jaata.

Assign Target

Button

Target value save karta hai + covered logon ko notification. (Isse pehle "CTC preview" sirf suggestion deta hai.)

⚠ Dhyaan dene wali baat — Incentive engines

Do naye engines (Slab aur Formula) dono Payroll.incentive ko replace karte hain, jodte nahi. Matlab jo Run aakhri me chalega, wahi payroll ka incentive number ban jaayega. Dono ek saath chahiye ho to formula me hi mila ke rakhna behtar.
